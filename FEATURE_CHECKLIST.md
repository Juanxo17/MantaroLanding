2️⃣  CONFIGURAR API KEY
    🔑 Obtener en: https://console.anthropic.com
    📝 Colocar en: .env.local
    
    VITE_ANTHROPIC_API_KEY=tu_api_key_aqui

3️⃣  CORRER LOCALMENTE
    ```bash
    cd "Mantaro web/m"
    npm install  # (ya hecho)
    npm run dev
    ```

    🌐 http://localhost:5173
